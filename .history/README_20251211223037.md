# 中国象棋AI训练系统

基于10层神经网络的自我对弈训练系统。

## 功能特性

- **10层深度神经网络**：使用深度神经网络评估棋盘局面
- **自我对弈训练**：自动生成高质量训练数据
- **人机对战**：支持玩家与AI对战
- **性能监控**：实时显示CPU、内存使用情况
- **自动调度**：支持GitHub Actions定时运行
